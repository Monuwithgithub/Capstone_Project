# Capstone_Project : # 🍕 Pizza Sales Analysis Project

## 📌 Project Overview
This project analyzes pizza sales data to uncover key business insights such as revenue trends, customer behavior, and product performance. The goal is to help the business make data-driven decisions to improve sales and profitability.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-selling and low-performing pizzas
- Understand customer ordering patterns
- Find peak sales hours and days
- Provide actionable business insights

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel (Data Cleaning & Initial Analysis)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)
- Python (EDA )

## 📊 Key Metrics (KPIs)
- Total Revenue  
- Total Orders  
- Total Pizzas Sold  
- Average Order Value  
- Average Pizzas per Order  

---

## 📈 Dashboard Features
- Sales Trend Analysis (Daily/Monthly)
- Revenue by Category
- Top 5 Best-Selling Pizzas
- Bottom 5 Low-Performing Pizzas
- Sales by Pizza Size
- Orders by Hour and Day

---

## 🔍 Key Insights
- Peak sales occur during weekends and evenings  
- Large size pizzas generate the highest revenue  
- A small number of pizzas contribute to most of the revenue  
- Certain pizzas consistently underperform and need attention
